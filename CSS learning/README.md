# 🎨 CSS Learning Journey

This folder documents my journey of learning CSS — focusing on styling, layout techniques, and building responsive user interfaces.

## 📘 Topics Covered

* CSS Basics (selectors, properties, colors)
* Box Model (margin, padding, border)
* Typography and text styling
* Flexbox
* CSS Grid
* Positioning (static, relative, absolute, fixed)
* Responsive Design (media queries)
* Transitions and animations

## 🛠 Practice

The `projects/` folder contains small projects to apply concepts, such as:

* Styled landing pages
* Navigation bars
* Responsive layouts
* UI components

Concepts, examples, and key learnings are documented in `notes.md`.
